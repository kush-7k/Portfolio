Wedding Website

  • Tech Stack:Node, HTMl, CSS, Javascript(Angular, Express), Bootstrap, GoogleAPIs
  • Github URL: Project Link

Developed a website for my brother's wedding with features including:

Uploading and submitting multiple media documents instantly to a provided Google Drive link using Google APIs.

User input for email address to send a thank you email after submitting media, along with the Google Drive link for viewing captured moments.

Rent this website option available at a fixed price.

User input for name and email to inquire about renting the website.

notification of rental enquiries, while users receive rates and discounts via email
