Java Swordigo Terminal Game

  • Tech Stack: Java, IDE(IntelliJ idea), Testing, Algorithm, OOP, Version Control
  • Github URL: Project Link

The Java Swordigo Game for the terminal offers an immersive and thrilling adventure where players engage in epic battles against the computer opponent. Set in a fantasy world, players and the computer each start with a maximum of 100 hit points. Using strategic attacks and cunning maneuvers, players must strategically choose their moves to outwit and defeat their opponent before they themselves are defeated. The game features a turn-based combat system, allowing players and the computer to take turns unleashing attacks. Each player has access to a variety of standard attacks as well as awe-inspiring special attacks, adding an element of surprise and excitement to the battles. With only a single chance to attack per turn, players must carefully plan their moves and anticipate their opponent's actions to emerge victorious. With its engaging gameplay and challenging battles, the Java Swordigo Game promises hours of thrilling entertainment for players seeking adventure and excitement in the world of terminal-based gaming.
