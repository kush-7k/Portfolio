Attendance System

  • Tech Stack: Java, SceneBuilder, MVC architecture, JavaFX, Scene
  • Github URL: Project Link

The student attendance system developed with Java MVC (Model-View-Controller) architecture and Scene Builder offers a structured and user-friendly approach to tracking student attendance. In this system, the Model represents the data layer, handling student information, attendance records, and database interactions.

The View encompasses the user interface components, designed using Scene Builder to create visually appealing and intuitive interfaces for users to interact with.

The Controller acts as the intermediary, managing user inputs, processing data, and updating the View accordingly. Through this MVC design pattern, the system ensures separation of concerns, scalability, and maintainability, providing educators with a reliable tool to efficiently monitor and manage student attendance records.
