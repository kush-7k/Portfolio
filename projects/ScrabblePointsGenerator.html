ScrabblePoints Generator

  • Tech Stack: Java, FXML, JavaFX, IDE(intelliJ Idea), MVC, Executable JAR, JUnit Tester
  • Github URL: Project Link

The Scrabble Points Generator is a Java application designed to assist players in calculating the point values of words according to the rules of the popular word game Scrabble. Implemented with JavaFX for the graphical user interface (GUI), the application allows users to input words and instantly generate their corresponding point values based on Scrabble's scoring system, which assigns different point values to each letter. The application follows the Model-View-Controller (MVC) architecture, separating the data logic (Model) from the presentation (View) and user interactions (Controller).

Using Scene Builder, developers can easily design and customize the application's interface with drag-and-drop functionality. This user-friendly tool enhances the user experience, enabling seamless navigation and efficient word entry. Additionally, the application is packaged as an executable JAR file, allowing users to conveniently run it on their local machines without requiring additional dependencies. With its intuitive interface and accurate point calculation functionality, the Scrabble Points Generator is a valuable tool for Scrabble enthusiasts looking to enhance their gameplay experience.
